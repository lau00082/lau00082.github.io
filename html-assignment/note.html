Heading
- (h1, h2, h3, h4, h5, h6) in HTML are used to define different levels of headings within a document. They provide structure, hierarchy, and help with accessibility and SEO.
Paragraph
- is an HTML tag used for defining paragraphs of text. It creates block-level elements that start on new lines and span the full width of their container.
Anchor
- is an HTML tag used for creating hyperlinks. It's called the anchor tag and is used with the href attribute to specify the destination URL
Description list
- is an HTML tag used to create description lists, consisting of term-description pairs. It's often used for glossaries or metadata.
footer
- is an HTML tag used to define the footer section of a webpage. It usually contains copyright information, contact details, or related links.
